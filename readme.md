# Personal NixOS Configuration

This repository contains **my personal NixOS configuration**.

It is **not intended to be used verbatim on other systems**.

## Reuse

You are **welcome and encouraged** to:

- Read the configuration for inspiration
- Copy individual snippets, modules, or ideas
- Adapt parts of it into your own NixOS setup
- Learn from patterns used here

However, you **should not** use this repository as-is as your own system configuration.

If you do reuse parts of it, **make them your own** and understand what they do.
